1. User:
   - UserID (Primary Key)
   - Username
   - Email
   - Password
   - Profile Information (Bio, Location, etc.)
   - skills (array 1st index= mainskill)
   - Join Date
   - Active
   - Last login
   - Profile Image

2. Tweet:
   - TweetID (Primary Key)
   - UserID (Foreign Key referencing User)
   - Content (obect 2keys 2key=media i.e(list))
   - Timestamp
   - Likes Count

3. Comment:
    - commentId (Primary key)
    - tweetId (foreigh key)
    - Content (obect 2keys 2key=media i.e(list))
    - Timestamp
    - Likes Count

3. Skill:
   - SkillID (Primary Key)
   - Skill Name
   - Description
   - Category

4. Course:
   - CourseID (Primary Key)
   - cousercode (primary key)
   - Title
   - Description
   - SkillID (Foreign Key referencing Skill)
   - InstructorUserID (Foreign Key referencing User)
   - Duration
